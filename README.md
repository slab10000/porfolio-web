# Portfolio Website

A modern, interactive portfolio website showcasing the work and expertise of Blas Moreno Laguna, a Computer Engineer and AI Engineer pursuing a Master's in Applied Artificial Intelligence at Purdue University Northwest.

## Overview

This portfolio website is a single-page application built with React and TypeScript, featuring a sleek Swiss design aesthetic with smooth animations and interactive elements. The site serves as a comprehensive showcase of professional experience, technical projects, skills, and educational background.

## Features

- **Hero Section**: Introduction with an animated astronaut theme and personal branding
- **Experience Timeline**: Detailed work history including roles at Santander/Openbank and NEC Labs/IMDEA Networks Institute
- **Projects Showcase**: Interactive project cards with detailed modals featuring:
  - ROS & Edge Computing Research
  - GPT from Scratch implementation
  - Roman Empire Counter Android App
  - AI & Image Recognition projects
  - Rocket Software development (STAR UC3M)
  - Sophra AI Writing Assistant
- **Skills Section**: Organized display of technical skills across languages, frameworks, AI/ML tools, and soft skills
- **Social Media Integration**: Links to TikTok, X (Twitter), and YouTube profiles
- **Responsive Design**: Fully responsive layout optimized for all device sizes
- **Interactive Animations**: Powered by Framer Motion for smooth transitions and engaging user experience
- **3D Background**: Star field background using Three.js for an immersive visual experience

## Technology Stack

- **Frontend Framework**: React 19 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom Swiss design theme
- **Animations**: Framer Motion
- **3D Graphics**: Three.js
- **Icons**: Lucide React
- **Architecture**: Component-based with clean separation of concerns

## Design Philosophy

The portfolio follows a Swiss design approach with:
- Clean, minimalist aesthetics
- High contrast and readability
- Bold typography
- Strategic use of color (Swiss lime accent)
- Focus on content and functionality
